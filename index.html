College Management System


Introduction

College Management System is a Collage themed Database created in MySQL and accessed with SQL. SQL is used in the accessing, updating, and manipulation of data in a database while MySQL is an RDBMS that allows keeping the data that exists in a database organized. SQL is a Structured Query Language and MySQL is a RDBMS to store, retrieve, modify and administrate a database

Step One:


1. First we create a Database named College Management System.

CREATE DATABASE CollegeManagementSystem;



Step Two:


2. Next we create tables in MySQL for our database. A total of 6 tables are created with unique names, list of columns, data types and keys.

Flip the images below to see a detailed description of each table.



Department Table



The Department Table consists for 2 columns.

1. "id" data type of INT and is also the primary key and
2. "name" data type of VARCHAR(50).

Course Table



The Course Table consists for 3 columns.

1. "id" data type of INT and is also the primary key,
2. "name" data type of VARCHAR(50) and
3. "deptId" an INT and also the foreign key.




Faculty Table



The Faculty Table consists of 4 columns.

1. "id" data type of INT and is also the primary key,
2. "firstname" data type VARCHAR(30),
3. "lastname" data type VARCHAR(50) and
4. "deptId" an INT and also the foreign key.

Faculty Course Table



The Faculty Course Table consists of 2 columns.

1. "facultyId" data type INT and
2. "courseId" data type INT and also the foreign key.




Student Table



The Student Table consists of 9 columns.

1. "id" data type of INT and is also the primary key,
2. "firstname" data type VARCHAR(30),
3. "lastname" data type VARCHAR(50),
4. "street" data type VARCHAR(50),
5. "streetDetail" data type VARCHAR(30),
6. "city" data type VARCHAR(30),
7. "state" data type VARCHAR(30),
8. "postalCode" data type CHAR(5) and
9. "majorId" data type INT and also the foreign key.

Student Course Table



The Student Course Table consists of 4 columns.

1. "studentID" data type INT and the foreign key,
2. "courseId" data type INT,
3. "progress" data type INT and
4. "startDate" data type DATE also a foreign key.
